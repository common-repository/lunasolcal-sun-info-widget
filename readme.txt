Wordpress Widget for showing the sunrise and sunset at a given location.

== Description ==
LunaSolCal - Sun Info Widget is a plugin that allows you to add a widget to your Wordpress site that displays the times of sunrise and sunset at a given location.

##How to add it to the sidebar:
* Go to Appearance > Widgets
* Press "+" to add a new block
* Find LunaSolCal Sun Info Widget (you might have to press "Browse all" to show all available Widgets)
* Add to a sidebar
* Enter a location name, latitude, longitude and select the time zone. You can find the latitude and longitude values of your city on it's Wikipedia page. 
* Press "Update" to save the changes

== Installation ==
1. Go to Plugins / Add New.
2. Search for LunaSolCal in the search box
3. The “LunaSolCal Sun Info Widget” plugin should appear. Click on Install.
